## Getting Started

### Prerequisites
node v18.14.0

### Installation
`git clone https://github.com/brian-thornton/axe-scoreboard`\
`cd axe-scoreboard`\
`npm install`\
`npm start`

### Notes
NOTE: All data is stored in browser localStorage. If you clear your browser cache all match history and player data will be LOST.
